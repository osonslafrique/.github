# OSONS L’AFRIQUE

**OSONS L’AFRIQUE** est un parti politique panafricain ivoirien engagé pour la souveraineté, la dignité, l’unité et la renaissance africaine.

Présidé par **ZOKOU Osso Paul**, le Parti porte une vision républicaine, humaniste et panafricaine fondée sur la responsabilité politique, la fraternité nationale, la cohésion sociale, la souveraineté des peuples africains et la construction d’une Afrique forte, organisée et respectée.

## Objet de cette organisation GitHub

Cette organisation GitHub constitue l’espace technique officiel destiné au développement, à la maintenance et à l’évolution des outils numériques du Parti, notamment le site officiel, les interfaces publiques, les documents techniques et les projets numériques associés.
